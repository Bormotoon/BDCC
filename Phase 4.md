*Любой граф, созданный моддером, после трансляции в код будет наследовать этот класс. Это даст моддерам доступ к безопасному API игры без возможности сломать ядро.*

class_name CrotchScriptBase extends RefCounted

## Базовый класс для всех сгенерированных CrotchCode скриптов (модов).
## Предоставляет безопасный API для вызовов из визуального графа.

var _is_cancelled: bool = false

## ШАГ 2: Транспайлер (Генератор Кода)

*Это ядро системы. Он принимает словарь (описание графа) и превращает его в строку GDScript.*

1. Создай файл `Systems/CrotchCode/CrotchTranspiler.gd`:

```gdscript
# Systems/CrotchCode/CrotchTranspiler.gd
class_name CrotchTranspiler extends RefCounted

## Преобразует граф визуальных блоков CrotchCode в нативный GDScript 2.0 код.

const HEADER = """# GENERATED BY CROTCHCODE TRANSPILER
extends CrotchScriptBase

func execute() -> void:
"""

## Принимает граф блоков и возвращает текст готового .gd скрипта
func transpile(blocks: Dictionary, start_node_id: String) -> String:
    var generated_code: String = HEADER
    var current_node_id: String = start_node_id
    var indent: String = "\t"

    # Упрощенный линейный обход графа (в реальности будет обход дерева с ветвлениями)
    while current_node_id != "" and blocks.has(current_node_id):
        var block = blocks[current_node_id]

        # Генерация кода для конкретного типа блока
        var line := _generate_block_code(block)
        if line != "":
            generated_code += indent + line + "\n"

        # Защита от бесконечного цикла в сгенерированном коде (впрыскиваем await для while)
        if block.type == "while_loop":
            generated_code += indent + "\tawait cc_wait_seconds(get_tree(), 0.01)\n"

        current_node_id = block.get("next_id", "")

    return generated_code

## Транслирует конкретный блок в строку кода
func _generate_block_code(block: Dictionary) -> String:
    match block.type:
        "print":
            # Безопасно экранируем строки
            var text = str(block.get("text", "")).replace('"', '\\"')
            return 'cc_print("%s")' % text
        "give_item":
            return 'cc_add_item(target_entity, &"%s", %d)' % [block.get("item_id", ""), block.get("amount", 1)]
        _:
            return "# Блок неизвестного типа: %s" % block.type
```

## ШАГ 3: Компилятор в реальном времени

*Класс, который берет строку от транспайлера и превращает её в исполняемый объект во время игры.*

1. Создай файл `Systems/CrotchCode/CrotchCompiler.gd`:

```gdscript
# Systems/CrotchCode/CrotchCompiler.gd
class_name CrotchCompiler extends RefCounted

## Компилирует сгенерированный строковый код в рантайме.

## Компилирует код и возвращает экземпляр скрипта
func compile_and_instantiate(source_code: String) -> CrotchScriptBase:
    var script = GDScript.new()
    script.source_code = source_code

    var error = script.reload()
    if error != OK:
        push_error("CrotchCompiler: Ошибка компиляции сгенерированного кода! Код ошибки: ", error)
        push_error("Исходный код:\n", source_code)
        return null

    var instance = script.new()
    if instance is CrotchScriptBase:
        return instance
    else:
        push_error("CrotchCompiler: Сгенерированный скрипт не наследует CrotchScriptBase!")
        return null
```
